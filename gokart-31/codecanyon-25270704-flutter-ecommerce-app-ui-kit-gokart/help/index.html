B) Directories & File Uses

Our Design uses only assets & lib folder. So, don't touch any other folder for ui changes.

    • assets(Folder) - Put all Static images & other data (in Our Case images & json file) in this folder
    • lib(Folder) - Main folder that contains all the dart code
      • animation(Folder) - Put all Animation file in This folder
        • fadeIn.dart(File) - Fade in animation file that use in Login & Signup page.
        • slide_left_rout.dart(File) - Right to Left Page Rout animation file that use in various page.
      • AppTheme(Folder) - Put all Theme file in This folder
        • my_behaviour.dart(File) - Behaviour File of App.
      • functions(Folder) - Put all Functions file in This folder
        • passDataToProduct.dart(File) - Pass data to product page function.
      • Pages(Folder) - Contain All App Pages in this folder except main.dart
        • category(Folder) - Contain All Category related Pages in this folder
          • top_offers_pages(Folder) - Contain All Top Offer Page Element. Like sort & filter row, get grid data, etc.
            • filter_row.dart(Page) - Contain filter row code (Sort & Filter) that show in women_wears.dart.
            • get_products.dart(Page) - Get Product Data code that show in women_wears.dart.
            • women_wears.dart(Page) - Product page that show all product data that comes from filter_row.dart & get_products.dart.
          • filter.dart(Page) - Filter Page Code in this file.
          • top_offers.dart(Page) - Category page that show when user click on category.
        • faq_and_about_app(Folder) - Contain FAQ & About App Page in this folder
          • about_app.dart(Page) - About App Page.
          • faq.dart(Page) - FAQ Page.
        • home_page_component(Folder) - Contain All the Component that use in home.dart Page in this folder
          • best_deal.dart(Page) - Contain Code for best deal section of home page.
          • best_of_fashion.dart(Page) - Contain Code for best of fashion section of home page.
          • best_offer_grid.dart(Page) - Contain Code for best offer section of home page.
          • block_buster_deal.dart(Page) - Contain Code for Block Bustor Deal section of home page.
          • category_grid.dart(Page) - Contain Code for Category Grid section of home page.
          • drawer.dart(Page) - Contain Code for Drawer section of home page.
          • featured_brands.dart(Page) - Contain Code for Featured Brands Slider section of home page.
          • top_seller_grid.dart(Page) - Contain Code for Top Seller section of home page.
          • womens_collection.dart(Page) - Contain Code for Womens Collection section of home page.
        • my_account(Folder) - Contain All the Pages of My Account & Account Setting in this folder
          • account_setting.dart(Page) - Account Setting Page
          • my_account.dart(Page) - My Account Page
        • order_payment(Folder) - Contain Delivery Address & Payment in this folder
          • delivery_address.dart(Page) - Delivery Address Page
          • payment.dart(Page) - Payment Page
        • product(Folder) - Contain all component & page related product page
          • all_rating.dart(Page) - All rating page
          • get_similar_product.dart(Page) - Get similar product and return it to product_details.dart
          • product_details.dart(Page) - All component of product page (Like Slider, title, price, size, color, etc.) are in this page and return Scaffold body to product.dart
          • product_size.dart(Page) - Product Size & Color Component
          • product.dart(Page) - Main Product Page
          • rating_row.dart(Page) - Rating row contains product rating and total rating
        • cart.dart(Page) - Cart page
        • forgot_password.dart(Page) - Forgot Passsword page
        • home.dart(Page) - Home Page
        • login.dart(Page) - Login page
        • my_orders.dart(Page) - My Orders page
        • notifications.dart(Page) - My Notification page
        • search.dart(Page) - Search page
        • signup.dart(Page) - Signup page
        • splashscreen.dart(Page) - Splashscreen page
        • wishlist.dart(Page) - Wishlist page
    • main.dart - Entry Point of Our App.
  • pubspec.yaml(File) - Specification file (All the package and local assets (Like Images, Fonts, etc.) we use in this application is specify in this file.).
